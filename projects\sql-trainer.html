业务背景

掌握 SQL 不只是会写语法，还需要理解指标定义、数据粒度与分析动作。传统学习材料常把这三者拆开，练习结果很难映射到实际工作情境。

产品设计

我把练习题目、模拟数据、查询编辑器和结果表格集中在一个页面，让使用者在同一界面完成「理解指标 → 编写 SQL → 运行验证 → 复盘口径」的完整循环。

选择练习题查看数据表结构编写 SQL运行查询复盘口径

在线体验

以下练习器在浏览器中运行真实 SQL 查询（基于 AlaSQL 引擎）。数据为模拟业务场景，覆盖 DAU、GMV、留存率等常用互联网指标。

练习题目

01
日活用户数 (DAU)
按日期统计每天的活跃用户数
02
各渠道 GMV
按注册渠道汇总已支付订单金额
03
品类营收排名
找出营收 Top 5 的商品品类
04
用户下单转化率
计算注册用户中有过下单行为的比例
05
次日留存用户
找出注册后次日仍有登录的用户
